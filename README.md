# Website Thương mại điện tử

## Mô tả dự án
Dự án là một **website thương mại điện tử hoàn chỉnh**, tập trung vào việc xây dựng **Backend mạnh mẽ** và **giao diện người dùng truyền thống** dựa trên Java.  
Hệ thống được thiết kế để quản lý **Người dùng, Sản phẩm, Đơn hàng**, đồng thời đảm bảo **bảo mật và phân quyền** chặt chẽ.

### Các tính năng chính
- Quản lý sản phẩm, người dùng và đơn hàng với các dịch vụ CRUD
- Phân quyền chi tiết cho Admin và User
- Giao diện JSP kết hợp Bootstrap, responsive cho các trang giỏ hàng, thanh toán
- Hỗ trợ chức năng đăng ký/đăng nhập, giỏ hàng và thanh toán

## Công nghệ sử dụng
- **Backend:** Java, Spring Boot, Spring Security, RESTful API  
- **Frontend:** JSP, Bootstrap  
- **Database:** MySQL  

## Thành tựu nổi bật
### 1. Phát triển Hệ thống Lõi
- Triển khai các dịch vụ CRUD cốt lõi cho **Người dùng, Sản phẩm và Đơn hàng**
- Đảm bảo **hiệu suất cao** và **bảo mật**

### 2. Bảo mật Hệ thống & Phân quyền
- Sử dụng **Spring Security** để triển khai **Phân quyền chi tiết**
- Kiểm soát quyền truy cập của **Admin** và **User** đối với các tài nguyên và API quan trọng

### 3. Xử lý Logic & Giao diện
- Phát triển **logic nghiệp vụ phức tạp** ở Backend
- Sử dụng **JSP + Bootstrap** xây dựng giao diện cơ bản như giỏ hàng, trang thanh toán  
- Giao diện **responsive**, thân thiện với người dùng

### 4. Quản lý Chức năng Chính
- Chịu trách nhiệm hoàn toàn cho các module:
  - **Giỏ hàng**
  - **Thanh toán**
  - **Đăng ký/Đăng nhập**
